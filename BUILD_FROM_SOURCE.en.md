## Build from Source (Alternative)

### Prerequisites
- Rust (edition 2024) and Cargo.

### Build and Run

Run directly:

```bash
cargo run
```

Or build release:

```bash
cargo build --release
```

Binary output:

```text
target/release/telegram-buktikanbot
```

Run the built binary:

```bash
./target/release/telegram-buktikanbot
```
